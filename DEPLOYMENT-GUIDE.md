## 🆘 TROUBLESHOOTING

### El sitio no aparece después de 5 minutos
1. Verifica que "GitHub Actions" está seleccionado en Settings → Pages
2. Revisa que la rama correcta está configurada
3. Mira "Actions" para ver si el workflow tiene errores

### Las páginas no cargan completamente
1. Verifica que los archivos HTML están válidos
2. Revisa la consola del navegador (F12) para errores
3. Limpia caché del navegador

### Los estilos CSS no se ven
1. GitHub Pages puede tener delay de caché
2. Espera 5-10 minutos más
3. Intenta acceso incógnito para evitar caché local
