# origen-wordpress

Datos originales exportados del WordPress. **No se modifican**, son la fuente.

- `export/` → el archivo Export XML (WXR). Ej: `nanonex.WordPress.2026-06-01.xml`
- `uploads/` → copia de `wp-content/uploads` (las imágenes del sitio)

Una vez subido esto, se procesará para generar la web estática en `../web-html/`.
