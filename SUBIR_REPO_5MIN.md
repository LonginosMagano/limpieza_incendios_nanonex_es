🆘 SOLUCIONAR ERRORES
═══════════════════════════════════════════════════════════════════════════════

Error: "Username for 'https://github.com': No such device or address"
→ Problema: Contraseña incorrecta o token expirado
→ Solución: Crea nuevo Personal Access Token en https://github.com/settings/tokens

Error: "fatal: could not read Username for 'https://github.com'"
→ Problema: Sin conexión a internet o GitHub caído
→ Solución: Verifica conexión: ping github.com

Error: "Permission denied (publickey)"
→ Problema: Usando SSH en lugar de HTTPS
→ Solución: Usa HTTPS en lugar de SSH
→ Comando: git remote set-url origin https://github.com/LonginosMagano/limpiezaincendiosnanonexmadrid.git

Error: "Your branch is ahead of 'origin/master' by X commits"
→ Problema: Cambios locales no pusheados
→ Solución: git push origin master (con tu token)
