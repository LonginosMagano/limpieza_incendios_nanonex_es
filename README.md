# limpieza_incendios_nanonex_es

Repositorio del proyecto **Limpieza Incendios Nanonex (ES)**.

## Descripción

Este repositorio está preparado para alojar los archivos y la documentación
del proyecto. Añade aquí tu código, datos y documentos.

## Estructura del repositorio

```
.
├── docs/        # Documentación del proyecto
├── src/         # Código fuente
├── data/        # Datos y recursos
└── README.md    # Este archivo
```

## Cómo subir archivos

1. Copia tus archivos en la carpeta correspondiente (`docs/`, `src/` o `data/`).
2. Añádelos al control de versiones:

   ```bash
   git add .
   git commit -m "Añadir archivos"
   git push
   ```
